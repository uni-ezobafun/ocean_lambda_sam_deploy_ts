# ocean_lambda_sam_deploy_ts

aws-sam + cloudformation で、nodejs ランタイム(typescript)の lambda と apigw を運用

### setup

```
$ git clone https://github.com/uni-ezobafun/ocean_lambda_sam_deploy_ts.git
$ cd ocean_lambda_sam_deploy_ts
$ npm i
$ npm run build-dev
```
